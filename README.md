# Kelly Z 的个人博客

这是一个基于 [Hexo](https://hexo.io/) 框架和 [Butterfly](https://github.com/jerryc127/hexo-theme-butterfly) 主题搭建的个人博客。

## 🔗 访问地址

[https://firsticket.github.io](https://firsticket.github.io)

## 🛠️ 技术栈

- **框架**: Hexo
- **主题**: Butterfly
- **部署**: GitHub Pages + GitHub Actions
- **评论系统**: Utterances (基于 GitHub Issues)


## ✍️ 如何更新博客

1. 在本地执行 `hexo new "文章标题"` 创建新文章。
2. 编辑完成后，执行 `hexo clean && hexo generate` 生成静态文件。
3. 使用 `git add .`、`git commit -m "更新说明"` 和 `git push` 推送到 GitHub。
4. GitHub Actions 会自动构建并部署。
